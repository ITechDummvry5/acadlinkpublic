ACADLINK – Scholarship Finder

ACADLINK is a simple web-based scholarship finder designed to help students easily search and filter available scholarship opportunities. The system allows users to search using keywords, filter by academic strands such as STEM, ABM, HUMSS, TVL, and GAS, and check scholarship availability based on location, deadline, and open/closed status.

Built using HTML, CSS, and JavaScript, ACADLINK operates without a backend, making it lightweight, fast, and easy to deploy online. The platform centralizes scholarship listings from different organizations and institutions into one accessible system, reducing the time and effort students spend searching for opportunities.

Dynamic Search

Instant keyword-based scholarship search with loading animation feedback.

Strand Filtering

Filter scholarships by STEM, ABM, HUMSS, TVL, GAS and more.

Responsive Design

Clean scholarship cards with official application links and deadline display.

AI-Ready Tools

ACADLINKS Intelligence prepares the system for future AI integration.
